# Benildo's Site

WELCOME TO MY HOMEPAGE

Este é meu site pessoal em HTML puro, inspirado na estética da web dos anos 80 e 90.
Nada de frameworks modernos, nada de CSS avançado.
Apenas HTML simples, tabelas, fontes clássicas e muito verde no preto.

## Sobre o site

- HTML cru
- Visual estilo terminal
- Layout feito com tabelas
- Uso de tags clássicas como <center> e <font>
- Contador de visitantes por imagem
- Estilo pessoal, simples e nostálgico

## Conteúdo

- Apresentação pessoal
- Atividade profissional
- Links úteis
- Contato
- Contador de visitantes
